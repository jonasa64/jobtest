How to use

All users

  • request type GET
  • Url http://localhost:5000/users/
  • 404 page is return if the url is invalid

one users

  • request type GET
  • Url http://localhost:5000/users/:id
  • id is type int
  • 404 page is return if the url is invalid

insert new user

  • request type POST
  • Url http://localhost:5000/users/insert
  • 404 page is return if the url is invalid

delete user

  • request type delete
  • Url http://localhost:5000/users/:id
  • id type int
  • 404 page is return if the url is invalid

edit user

  • request type put
  • Url http://localhost:5000/users/edit/:id
  • id type int

    • params

    • firstname string
    • lastname string
    • email string
    • password string
  • 404 page is return if the url is invalid
